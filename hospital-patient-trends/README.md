# 🏥 Suffolk County NY Hospital Patient Trends 2023

## Overview
Analyzed 160,396 hospital discharge records from Suffolk County, NY using real SPARCS De-Identified state data from the New York State Department of Health.

## Tools Used
- Google Sheets (data cleaning, pivot tables)
- Tableau Public (dashboard visualization)

## Key Findings
- **Stony Brook University Hospital** had the most admissions (34,101)
- **70+ age group** had the highest admission rate (61,062)
- **Top diagnosis:** Liveborn (13,530), followed by Septicemia (12,011) and Heart Failure (4,451)
