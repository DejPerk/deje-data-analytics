## Projects

### 🏥 Suffolk County NY Hospital Patient Trends 2023
Analyzed 160,396 hospital discharge records from Suffolk County, NY using real SPARCS De-Identified state data.

**Tools:** Google Sheets, Tableau Public  
**Key Findings:**
- Stony Brook University Hospital had the most admissions (34,101)
- 70+ age group had the highest admission rate (61,062)
- Top diagnosis: Liveborn (13,530), followed by Septicemia (12,011)

🔗 [View Live Tableau Dashboard](https://public.tableau.com/app/profile/deje.perkins/viz/SuffolkCountyHospitalPatientTrends2023/Dashboard1)

### 📦 Inventory / Supply Chain Analysis
Analyzed inventory and sales trends across five Las Vegas retail locations to identify stock issues and revenue patterns.

**Tools:** Google Sheets, Excel, Pivot Tables, Dashboard Visualization  
**Key Findings:**
- Vegas Strip location generated the highest revenue
- Vape products outsold cigars/cigarettes significantly
- Multiple stores had recurring low-stock alerts impacting sales opportunities
