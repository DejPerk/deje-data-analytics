# Inventory / Supply Chain Analysis

## Overview
This project analyzes inventory and sales trends across five Las Vegas retail locations.

## Tools Used
- Google Sheets / Excel
- Pivot Tables
- Dashboard Visualization
- Data Cleaning

## Key Insights
- Vegas Strip generated the highest revenue
- Certain vape products sold significantly more than cigars/cigarettes
- Multiple stores had recurring low-stock alerts
- Inventory shortages impacted sales opportunities

## Files Included
- Raw dataset (.xlsx)
- Revenue by Store dashboard
- Top-Selling Products dashboard
- Low Stock Alerts dashboard
- Summary dashboard
