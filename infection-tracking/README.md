# 🦠 NY State Hospital Acquired Infection Tracking 2009-2023

## Overview
Analyzed hospital-acquired infection (HAI) data across New York State acute care facilities from 2009-2023 using official NYS Department of Health reporting data.

## Tools Used
- Google Sheets (data cleaning, pivot tables)
- Tableau Public (dashboard visualization)

## Key Findings
- **CDI Hospital Onset** was the most common infection type with 1,835 cases
- **Maimonides Medical Center** had the highest total infections observed
- **Infections peaked in 2010** (613) and declined significantly to just 28 by 2023
- Overall downward trend reflects improved infection prevention practices statewide
